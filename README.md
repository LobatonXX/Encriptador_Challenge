# Encriptador_Challenge
# Descripción
Encriptador de Texto es una aplicación web que permite cifrar y desencriptar texto utilizando un algoritmo sencillo de sustitución. 
# Requisitos
- 1. Sólo Letras Minúsculas: Solo debes ingresar minusculas para que la aplicacion funcione correctamente.
- 2. Sin acentos ni caracteres especiales: No se deben utilizar letras con acentos ni caracteres especiales.
- 3. Encriptación y Desencriptación: Debe ser posible tanto cifraR un texto como decifrar el texto a la palabra original.
# Caracteristicas
Encriptación : Convierte letras específicas en secuencias de caracteres.
Desencriptación : Convierte secuencias de caracteres en letras originales.
Interfaz de usuario : Limpia y fácil de usar con mensajes claros en caso de entradas vacías o incorrectas.
